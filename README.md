# 모임중심찾기서비스 - 잼잼

## 📖Description

모임 참가자들의 위치를 기반으로 최적의 중심 장소를 찾고, 게임 요소를 통해 모임 장소를 변경하는 서비스입니다. 중심 장소 찾기와 게임을 통한 장소 변경으로 모임의 편의성과 재미를 모두 제공합니다.

바로가기 👉 [jjam.shop](https://jjam.shop/)
