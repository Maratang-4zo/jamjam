# 빌드 및 배포 정리 문서

## 사용한 JVM
- **JAVA**: amazon-corretto-21.0.4.7.1-linux-x64
- **JVM**: JVM 21

## 웹서버
- **Spring Boot**: 내장 Tomcat 사용
- **Nginx**: 정적 파일(React) 제공 및 프록시 서버로 사용 (SSL 설정 포함)

## WAS 제품 등의 종류와 설정 값
- Spring Boot 내장 Tomcat 사용
- Nginx를 프록시 서버로 설정하여 Spring Boot로 요청 전달 (도메인: jjam.shop, SSL 설정 포함)
